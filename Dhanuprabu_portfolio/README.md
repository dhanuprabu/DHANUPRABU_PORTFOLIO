# Dhanuprabu J — Portfolio

Personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Structure

```
.
├── index.html      # Main page
├── style.css       # All styles
├── script.js       # Interactivity (nav, video, animations)
└── assets/
    ├── profile.jpg               # Profile photo
    ├── hero-video.mp4            # Hero background/card video
    └── Dhanuprabu_J_Resume.pdf   # Downloadable resume
```

## Run locally

No build step needed — just open `index.html` in a browser, or serve it:

```bash
npx serve .
```
